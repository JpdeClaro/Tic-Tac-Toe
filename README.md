# tic-tac-toe 

The tic-tac-toe is played on a 3x3 grid. Each player represents based on whose turn first (X - player) only 2 players (X, O) can play this game. There are eight possible winning combinations displayed on the App.jsx. Lastly, the return statement render the boardgame which also display the turns and winner of this game.
